## Venv
*remember to use a python venv, requirements.txt is provided*
```python
python3 -m venv .venv
source .venv/bin/activate
```

When done:
```python
deactivate
```

## Requirements
Whilst the virtual envoirnment *(venv)* is active;
```python
pip3 install -r requirements.txt
```
